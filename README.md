ZaibatsuPass
============

ZaibatsuPass is an NFC transit card reading application.

It currently supports:

* ORCA (Greater seattle area)

In future, it should support

* Clipper (San Francisco, CA)
* Other MiFare DESfire
* FeliCa cards (SUICA, etc)
